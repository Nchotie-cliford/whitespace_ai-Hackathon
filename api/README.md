# Data Architect (API Module)

This module is responsible for bridging the gap between the chaotic simulation and the HERO software environment. 

## Responsibilities
- Executing GraphQL scripts to ingest data base configurations.
- Abstracting API calls (`queries` and `mutations`) so other modules don't deal directly with external requests.
- Seeding the Chaos Environment (e.g. creating base Contacts, Projects, and Tasks).
